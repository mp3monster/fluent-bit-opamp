# opamp

Recommended Python project scaffold using a modern `src/` layout.

## Quickstart

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e '.[dev]'
pytest
opamp
```

## Structure

- `src/opamp/` - application package
- `tests/` - test suite
- `docs/` - documentation
- `scripts/` - utility scripts
